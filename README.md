# Sound Cloud Clone

> 2022 Practical Coding React-Node TS Web Project

## 기능

- Music 검색
- 좋아요 / 좋아요 취소
- Like한 Music들 확인하기

## 실행 방법

```
$ git clone https://github.com/jukangpark/SoundCloud_Clone.git
$ npm install
$ cd server
$ npm run dev
```
